Brief

This project explores how the idea of rate coding can help scientists decode neural spike trains from brain implants. It analyzes data from a cat's visual cortex and shows how the scale at which one averages signals over time can radically affect the performance of models.

This project starts with pre-coded spike-trains. Neural spikes have to be detected and time-stamped before further analysis can proceed. I delivered two lectures on the topic of neural spike detection on YouTube. You can find Lecture I here and Lecture II here.

repository PDF article

Tools

Topics

Spatiotemporal Modeling

Brief

This project crafts a research oriented database from the valueable but hard to work with PLUTO data sets. The final database enables researchers to make queries about trends in space and time, especially pertaining to residential buildings in the New York City area. Additionally, it produces tables that contain lagged longitudinal data on percent changes in building values per unit with four geo-spatial neighbors per record.
